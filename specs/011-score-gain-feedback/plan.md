# Implementation Plan: Score Gain Feedback

## Summary
Implement immediate visual feedback for scoring moves by displaying a floating total point indicator (+N) and highlighting all tiles in the scoring connection with a Gold/Yellow glow.

## Technical Context

**Language/Version**: TypeScript 5.4+ (Next.js 14+)  
**Primary Dependencies**: React 18, Tailwind CSS 4.x, Lucide React, PeerJS  
**Storage**: N/A (Client-side game state)  
**Testing**: Vitest  
**Target Platform**: Web (Vercel-native)
**Project Type**: Web Application (P2P Game)  
**Performance Goals**: <100ms feedback latency, 60 fps animations  
**Constraints**: WCAG AA (4.5:1 contrast), Monotone aesthetic, P2P synchronization  
**Scale/Scope**: 2-player P2P
