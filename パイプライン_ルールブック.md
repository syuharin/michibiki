# みちびき - ルールブック

## 📖 概要

2人対戦のタイルプレイスメントゲーム。罫線パターンが描かれたタイルを交互に盤面に配置し、自分の色のパイプをつなげて得点を獲得する。相手のタイル重ね置き（逆転用）を活用して、相手の得点機会を邪魔しながら、自分のパイプを完成させることが目標。

---

## 🎮 ゲームの準備

### プレイヤー
- 2人

### 盤面
- 6×6のグリッド（36マス）

### タイル
各プレイヤーは以下のタイルを同じセットで保有します：

| タイル | 個数 | 説明 |
|--------|------|------|
| `━` | 2 | 横つなぎ（左右に接続点）※1枚は逆転用 |
| `┃` | 1 | 縦つなぎ（上下に接続点） |
| `┗` | 1 | 左上コーナー（上右に接続点） |
| `┛` | 1 | 右上コーナー（上左に接続点） |
| `┓` | 1 | 右下コーナー（下左に接続点） |
| `┏` | 1 | 左下コーナー（下右に接続点） |
| `┫` | 1 | T字型（上下左に接続点） |
| `┣` | 2 | T字型（上下右に接続点）※1枚は逆転用 |
| `┳` | 1 | T字型（下左右に接続点） |
| `┻` | 1 | T字型（上左右に接続点） |
| `╋` | 1 | 十字（上下左右に接続点） |

**合計：各プレイヤー13枚のタイル**

### 山札
- 各プレイヤーの13枚タイルを混ぜたもの
- 盤面に配置されたタイル数 + 手札の枚数 = 常に13枚

### 手札
- ゲーム開始時：各プレイヤーは山札から3枚を引く

---

## 🔄 ゲームの流れ

### 1. ゲーム開始
1. 6×6の空の盤面を用意
2. 各プレイヤーは山札から3枚ずつ引く
3. ゲーム外で先手・後手を決定
4. ゲーム上で先手・後手のプレイヤーを指定

### 2. ターンの進行
**先手プレイヤーから交互にターンを実行します。**

1. **タイル配置フェーズ**
   - 手札から1枚選び、**0°、90°、180°、270°のいずれかの向きに回転** させてから、盤面のいずれかのマスに配置
   - **重要**：盤面に配置した後は、そのタイルの向きを変更（回転）することはできない
   - **重ね置き**：逆転用タイル（`┣` または `━`）のみ、既に置かれているマスの上に重ねられる
   - 最大2枚までしか重ねられない

2. **補充フェーズ**
   - 山札から1枚引く（山札がなければ引かない）
   - 手札は常に0～3枚の範囲

3. **得点確認**
   - 配置したタイルが既存のパイプとつながり、新たにパイプが完成した場合、自動的に得点が計算される
   - **重要**：パイプは同じプレイヤーの色のタイル同士でのみつながる

### 3. ゲーム終了
- 全プレイヤーの手札が0枚になり、山札も空になったとき、ゲーム終了
- その時点での各プレイヤーの得点を集計
- **得点が高いプレイヤーが勝利**

---

## 📊 パイプとつながりの判定

### パイプとは
- 同じプレイヤーの色のタイル同士が、罫線の接続点で隣接しているもの

### つながりの判定ルール

**基本原則**：罫線の「端（接続点）」が実際に接続している必要があります。

#### 各タイルの接続点

| タイル | 接続点 |
|--------|--------|
| `━` | 左、右 |
| `┃` | 上、下 |
| `┗` | 上、右 |
| `┛` | 上、左 |
| `┓` | 下、左 |
| `┏` | 下、右 |
| `┫` | 上、下、左 |
| `┣` | 上、下、右 |
| `┬` | 下、左、右 |
| `┴` | 上、左、右 |
| `╋` | 上、下、左、右 |

#### 隣接の定義
- 縦横のみ（上下左右の4方向）
- 斜めはつながらない

#### つながる条件の例

```
✅ つながる例：

盤面（一部）
  ① ②
A ┫ ┃
B ┃ 

理由：
- ①A の┫：下に接続点あり
- ①B の┃：上に接続点あり
→ 上下で接続している

```

```
❌ つながらない例：

盤面（一部）
  ① ②
A ━
B ┃
C ┓

理由：
- ①A の━：接続点は左右のみ（下がない）
- ①B の┃：接続点は上下
→ つながっていない
```

### 得点の計算（累積加算方式）

このゲームでは、タイルを1枚配置するたびに得点が発生し、現在のスコアに**累積加算**されます。

- **加算スコア = 配置したタイルを含む、つながったタイルの総数**
- 1枚の孤立したタイルを置いた場合：**1ポイント** 加算
- 既に2枚つながっているパイプに3枚目を接続させた場合：**3ポイント** 加算（合計スコアはそれまでの累積 + 3）
- **重要**：同じパイプを伸ばし続けることで、既に置かれているタイルの分も「つながっている数」として再度カウントされるため、大きなパイプを作るほど得点が加速します。

### 重ね置きとパイプの関係

- **同じマスに2枚重ねられている場合**、上のタイルのみが得点判定に参加する
- 上に置いたタイルだけが「見える」状態になり、下のタイルはパイプ判定から除外される
- 相手のタイルに重ね置きすることで、相手のパイプを妨害できる

#### 例

```
盤面（一部）
   ① ②
A  ┣  ┫  （上：プレイヤーA、下：プレイヤーB）
   ────  （重ね置き）

見える盤面：
   ① ②
A  ┣  ┫

プレイヤーA の判定：
- ①A の┣（上層）：接続点は上下右
- ②A の┫：接続点は上下左
→ つながっている = 2ポイント

プレイヤーB の判定：
- ①B の━（下層）：接続点は左右
- ②B のタイルなし
→ つながっていない = 1ポイント
```

## 🎯 重ね置き（逆転用タイル）

### 重ね置きの仕組み

- **逆転用タイル（`┣` と `━` を1枚ずつ）のみ** が、既に盤面に置かれているタイルの上に重ねて配置可能
- その他のタイルは重ね置きできない（新しいマスにのみ配置可能）
- **同じマスには最大2枚までしか置けない**

### 重ね置き時の表示と判定

- **上のタイルが優先される**
  - 見た目には上に置いたタイルが表示される
  - パイプ判定では、**上のタイルの接続点のみが考慮される**
  - 下のタイルは完全に隠れて、パイプ判定に参加しない

### 重ね置きの活用

**邪魔する**：
- 相手が次にタイルを置きたい場所に、先に逆転用タイルを重ねる
- 相手のパイプ完成を妨害できる

**逆転する**：
- 相手が得点するはずだった場所に逆転用タイルを重ね置きして、パイプをつながりにくくする
- 相手の下層タイルを隠すことで、パイプの完成を防ぐ

**注意**：
- 逆転用タイルは**各プレイヤー1枚ずつ（`┣` と `━`）** のみ
- 使用すると補充されない（限られたリソース）
- 戦略的に使用することが重要

---

## ⚠️ 禁止事項

- 盤面外のマスにタイルを置く
- 同じマスに3枚以上のタイルを置く
- 逆転用タイル以外で重ね置きをする
- 一度配置したタイルを動かす、または向きを変える（回転させる）

## 🏆 勝敗の決定

- ゲーム終了時点で、**得点が高いプレイヤーが勝利**
- 同点の場合は「引き分け」

## 📝 ゲーム進行のコツ

### 攻撃的プレイ
- 相手のパイプを完成させそうな場所に、自分のタイルを配置して邪魔する
- 重ね置きを活用して、相手の得点機会を奪う

### 防守的プレイ
- 自分のパイプをコツコツつなげて、着実に得点を稼ぐ
- 相手の妨害に気をつけながら、確実な得点を狙う

### 逆転狙い
- 重ね置きを温存しておき、終盤で相手の大きなパイプ完成を邪魔する
- 相手の邪魔タイルを利用して、自分のパイプを完成させる

## 🎮 例題：累積スコアの加算例

### 例1：パイプを1枚ずつ伸ばす場合

1枚目、2枚目、3枚目と横に並べていくと、スコアは以下のように加算されます。

1.  **①A に ━ を配置**（1枚のみ）
    - 加算：1点
    - **累計：1ポイント**
2.  **②A に ━ を配置**（①Aとつながり、接続数は2枚）
    - 加算：2点
    - **累計：1+2 = 3ポイント**
3.  **③A に ━ を配置**（①A, ②Aとつながり、接続数は3枚）
    - 加算：3点
    - **累計：3+3 = 6ポイント**

このように、大きなパイプを作ることで得点が加速します。

### 例2：コーナーを含むパイプの接続

```
盤面（既に置かれている状態）
   ① ②
A  ━  ━  （2つつながっている）
```

ここにプレイヤーが **①B に ┃** を置いた場合：
- ①A の **━** とつながる
- 接続されたグループは：①B(┃)、①A(━)、②A(━) の **計3枚**
- **現在の累計スコアに 3ポイント 加算される**

### 例3：重ね置きによる妨害の例（スコア影響）

```
盤面（初期状態、下層）
   ① ②
A  ━  ━  （既にプレイヤーAが置いており、累計得点済み）
```

相手プレイヤーBが、**①A の上に逆転用タイルの ┃** を重ねた場合：
- プレイヤーAの既得スコアは**減らない**（累積加算のため）
- しかし、プレイヤーAが次に **③A に ━** を置いても、①Aの **━** が隠れているため、**②A とはつながらない**
- そのため、プレイヤーAの次の配置による加算スコアは **1枚分（1点）** に抑えられる

### つながらない例

```
盤面（一部）
   ① ②
A  ━  

B  ┃  

プレイヤーの判定：
- ①A の━：左右に接続点（下がない）
- ①B の┃：上下に接続点
→ つながっていない

結果：
- ①A = 1ポイント
- ①B = 1ポイント
（別々）
```
